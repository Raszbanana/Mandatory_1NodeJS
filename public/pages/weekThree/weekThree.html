Nodemon

What is Nodemon? As you might know, making changes in your project requires a restart before said changes will be visible. Manually this would mean having to type "node yourApp.js" repeatedly each time. This would be very frustrating when you are actively developing small changes in your project. However luckily there is a tool called Nodemon, which will actively listen for any changes in your project and restart your app everytime you make a change. To install Nodemon we use the npm installer:


$ npm install -g nodemon

Notice that we also type in "-g" this means that we are installing nodemon globally so that we wont have to install it for every new project we create. Once Nodemon is installed you simply run your app using Nodemon instead:


$ nodemon yourApp.js

If you want nodemon to not just restart on your app when changes are made to your app file, but also changes to html, css, files etc you then need to tell it to do so by creating a nodemon.json file in your root folder and type in something like this:


{ "ext": "html, css" }

Nodemon will now actively listen for changes for the type of files you specify

Sending Data

Assuming that you have already setup a simple express server and can receive a get request through the web. Now you want to also be able to send data in a request. Using express this is very simplified. An example from a class excercise with beers:


$ app.post("/beers", (req, res) => {
1. const beerToCreate = req.body;
2. beerToCreate.id = beerIdCounter++;
3. beers.push(beerToCreate);
4. res.send({ data: beerToCreate });

To explain in detail, we are creating a new const which will contain the properties of the request body(1.). We then assign the id to our global beerIdCounter(2.), to make sure that every beer added has a unique identifier. Once this is done we can now add the new beer to our array of beers(3.). We then send back the data of the newly created beer to confirm that it was created and added to our array(4.)

Loops

Working with loops in conventional java, we usually would do a typical for loop, which would increment to a set or x value. However this uses several lines of code. In javascript we have built-in function to handle whatever type of operation we might need to perform. For example purposes, lets assume we have an array of numbers:


$ const numbers = [1, 2, 3, 4];

If we wanted to increment or decrement every number, we would normally just write a for loop like the following:


$ for(let number of numbers){ number += 1; }

However this can be simplified by using the "forEach" method, which all arrays in javascript has access to:


$ numbers.forEach(number => number++);

One thing to note however, is that the "forEach" method does not return anything, it merely executes whatever you tell it to execute on all the elements inside the array. Should you wish to iterate through an array AND return something, you should utilize the "map" function:


$ numbers.map(number => { number++ return number; });

Lastly but also important we have the "filter" function. The filter() function creates a new array with all elements that pass the test implemented by the provided function. What this means is that it will return a new array which are the result of the parameters you set. For example if i wanted the array to only have numbers above 2:


$ const newNumbers = numbers.filter(number => number < 2);
