Loops

Working with loops in conventional java, we usually would do a typical for loop, which would increment to a set or x value. However this uses several lines of code. In javascript we have built-in function to handle whatever type of operation we might need to perform. For example purposes, lets assume we have an array of numbers:


$ const numbers = [1, 2, 3, 4];

If we wanted to increment or decrement every number, we would normally just write a for loop like the following:


$ for(let number of numbers){ number += 1; }

However this can be simplified by using the "forEach" method, which all arrays in javascript has access to:


$ numbers.forEach(number => number++);

One thing to note however, is that the "forEach" method does not return anything, it merely executes whatever you tell it to on all the elements inside the array
