About functions

To start off, there is the fact that a function can be named, or anonymous.


function AddArrowFunction(x,y) { return x+y } // Named

const addArrowFunction = (x,y) => x+y; // Anonymous

Notably is that assigning a variable to a function, still makes the function anonymous


Another important thing to note is callback and currying.
